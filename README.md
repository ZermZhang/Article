### 1.整体介绍
该文件夹下包含的是硕士毕业论文中所用到的用来求解四边形剖分下的多尺度有限元方法(FEM-MsFEM)的主要程序。
整体分成了两个部分：
* **OverSampling**文件夹下包含的是求解超样本基函数的程序。
* **cip_mulbase_J1**文件夹下包含的是FEM-MsFEM方法的主要程序。

### 2. **OverSampling**
该文件夹下包含的程序是用来求解在组合多尺度有限元方法所要用到的超样本基函数信息的。
* main文件夹
* comm文件夹
* dat文件夹
* produceexactsolution文件夹
